# About
This project was made as personal project to my portfolio. I wanted to create something that could be potentially useful for some people. You can add cities from all over the world and a time slot, where everyone is available. Then the App computes time slots when everyone is free and also interactive slider with visualization of time and wheter it is in the time slot.
